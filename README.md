## CSS Preprocessor(전처리기)

CSS로 개발을 하다보면 금세 코드가 지저분 해지는 경우와, 복잡한 코드를 수정이 필요할 때에는 일일이 찾아서 수정해야 하는 번거로움이 있습니다. 이러한 문제때문에 css 전처리기가 나오게 되었고, 이는 css가 동작하기 전에 미리 동작합니다. 대표적으로 Sass, Less, Stylus가 있습니다.
